# gocalc
A simple command line expression evaluator written in Go.

## Supported Operators

| Operator   |      Description   | Example     |
|:----------:|:-------------------|:------------|
| +          | Addition           | 1 + 2       |
| -          | Subtraction        | 1 - 2       |
| *          | Multiplication     | 10 * 20     |
| /          | Division           | 10 / 20     |
| ^          | Power              | 2 ^ 4       |
| ()         | Parentheses        | 2 * (1 + 3) |
